Functions¶

SGD_Light sgd_CreateDirectionalLight()¶

Create a new directional light.

A maximum of 4 directional lights are supported by the renderer. If you create more than that, the 4 with the highest priority will be used. See sgd_SetLightPriority.

SGD_Light sgd_CreatePointLight()¶

Create a new point light.

A maximum of 32 point lights are supported by the renderer. If you create more than that, the 32 with the highest priority will be used. See sgd_SetLightPriority. If there are more than 32 with the highest priority, the 32 nearest the camera will be used.

SGD_Light sgd_CreateSpotLight()¶

Create a new spot light.

A maximum of 16 spot lights are supported by the renderer. If you create more than that, the 16 with the highest priority will be used. See sgd_SetLightPriority. If there are more than 16 with the highest priority, the 16 nearest the camera will be used.
