SGD_Collider sgd_CameraPick(SGD_Camera camera, float windowX, float windowY, int colliderMask)¶

Pick first collider along ray passing from camera eye through window coordinates.

Note that the colliderMask parameter is a ‘bitmask’ value, where each ‘1’ bit in the value’s binary representation repesents a collider type you want to be included in the pick, allowing you to pick more than 1 collider type in a single pick.

For example, if you want the pick to include collider types 0 and 3, you would set bits 0 and 3 in the colliderMask, resuling in a value of 2^0 | 2^3 == 9.

If you want the pick to include all collider types, use a colliderMask of -1, a bitmask value with all bits set.
