Scene¶

The scene is a collection of entities stored as a directed, acyclic graph, and a numer of rendering properties.

The Scene rendering properties are:

  • Clear Color - a color value used to clear the scene. Use sgd_SetClearColor() to set this property.

  • Environment Texture - a cube or equirect 2d texture that represents the scene environment for the purpose of specular reflection when lighting the scene. Use sgd_SetEnvTexture() to set this property.

  • Ambient Light Color - a color value used to represent the scene’s diffuse ambient lighting. Use sgd_SetAmbientLightColor() to set this property.

To render a scene, use the sgd_RenderScene function.

Once the scene has been rendered, use the sgd_Present() function to copy the render result to the app window.

Example code¶

#include <sgd/sgd.h>

sgd_Init();

sgd_CreateWindow("Window",GetDesktopWidth()/2, GetDesktopHeight()/2, SGD_WINDOW_FLAGS_CENTERED);

sgd_SetClearColor(1, .5f, 0, 1);

while((sgd_PollEvents() & SGD_EVENT_MASK_CLOSE_CLICKED) == 0) {

    sgd_RenderScene();

    sgd_Present();
}

sgd_Terminate();

void sgd_ClearScene()¶

Destroy all active entities.

void sgd_ResetScene(SGD_Bool releaseAllHandles)¶

Reset scene to default state.

Destroys all active entities in the scene.

If the release all handles argument is true, sgd_ReleaseAllHandles is also called, allowing you to reset to the initial app state with a single command.
