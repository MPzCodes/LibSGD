Entity¶

An entity is a visual element in the scene. Every entity has the following properties:

  • Name: name of the entity, as read from file or set by user. Use sgd_SetEntityName() and sgd_GetEntityName() to get and set this property. Use sgd_FindEntityChild() to search an entity and its descendants for an entity with a specific name.

  • Position - position of the entity, stored as x, y, z coordinates. Use sgd_SetEntityPosition(), sgd_GetEntityX(), sgd_GetEntityY() and sgd_GetEntityZ() to set and get this property.

  • Rotation - rotation of the entity, stored as a 3 X 3 orthogonal, normalized ‘basis’ matrix. Use sgd_SetEntityRotation(), sgd_GetEntityRX(), sgd_GetEntityRY() and sgd_GetEntityRZ() to get and set entity rotation using euler angles in the range 0..360.

  • Scale - scale of the entity, stored as x, y, z scale values. Use sgd_SetEntityScale(), sgd_GetEntitySX(), sgd_GetEntitySY() and sgd_GetEntitySZ() to get and set this property.

  • Parent - reference to parent entity. Use sgd_SetEntityParent() and sgd_GetEntityParent() to set and get this property.

  • Children - child entities. Use sgd_GetEntityChildCount() and sgd_GetEntityChild() to visit the children of an entity.

  • Visible flag - flag indicating whther the entity is visible or not. Use sgd_SetEntityVisible() and sgd_IsEntityVisible() to set and get this property. For an entity to be rendered by sgd_RenderScene(), it and its ancestor’s visible flags must be set to true.

  • Enabled flag - flag indicating when the entity is enabled or not. Use sgd_SetEntityEnabled() and sgd_IsEntityEnabled() to set and get this property. For an entity to be collided with or picked, it and it’s ancestor’s enabled flags must be set to true.
