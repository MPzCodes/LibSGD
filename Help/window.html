Window¶

To create a window, use sgd_CreateWindow().

Types¶

enum SGD_WindowState¶

Window states.

Values:

enumerator SGD_WINDOW_STATE_CLOSED = 0¶

Window is closed.

enumerator SGD_WINDOW_STATE_MINIMIZED = 1¶

Window is minimized.

enumerator SGD_WINDOW_STATE_NORMAL = 2¶

Window is normal.

enumerator SGD_WINDOW_STATE_MAXIMIZED = 3¶

Window is maximized.

enumerator SGD_WINDOW_STATE_FULLSCREEN = 4¶

Window is fullscreen.

enum SGD_WindowFlags¶

Window flags.

Values:

enumerator SGD_WINDOW_FLAGS_NONE = 0x00¶

No special flags.

enumerator SGD_WINDOW_FLAGS_FULLSCREEN = 0x01¶

Create fullscreen window.

enumerator SGD_WINDOW_FLAGS_RESIZABLE = 0x02¶

Create resizable window.

enumerator SGD_WINDOW_FLAGS_CENTERED = 0x04¶

Create window centered on desktop.

typedef enum SGD_WindowState SGD_WindowState

Window states.

typedef enum SGD_WindowFlags SGD_WindowFlags

Window flags.

Destroy window.

Set window position.

Get window y.

Set window size.
