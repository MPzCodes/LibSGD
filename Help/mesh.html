Types¶

enum SGD_MeshFlags¶

Mesh flags.

Values:

enumerator SGD_MESH_FLAGS_NONE = 0¶

No special mesh flags.

enumerator SGD_MESH_FLAGS_TANGENTS_ENABLED = 1¶

Mesh contains materials with normal maps.

enumerator SGD_MESH_FLAGS_BLENDED_SURFACES = 2¶

Mesh contains materials that use alphaBlend blend mode.

typedef enum SGD_MeshFlags SGD_MeshFlags
